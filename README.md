# ZapretGUI

Открытый исходный интерфейс для работы с системой запрета информации в РОССИЙСКОЙ ФЕДЕРАЦИИ. Позволяет управлять запретными списками сайтов и сервисов, а также осуществлять функциональность для доступа к запрещенным ресурсам или сервисам.

## Быстрый старт

### Установка
1. Зарегистрируйтесь на официальном сайте проекта
2. Склонируйте репозиторий проекта на своё устройство
3. Запустите приложение и следуйте инструкциям на экране

### Первый запуск
```bash
# Клонирование и запуск
cd path/to/ZapretGUI
# Запуск приложения
```

## Системные требования

- Операционная система: Windows 10/11
- .NET Framework 4.8 или выше
- Свободное пространство программного обеспечения
- Интернет-соединение для обновлений

## Структура проекта

```
ZapretGUI/
├── App.xaml                   # Основной файл приложения
├── App.xaml.cs                # Код запуска приложения
├── MainWindow.xaml            # Основное окно приложения
├── MainWindow.xaml.cs         # Код основного окна
├── Views/                     # Представления и интерфейсы
├── ViewModels/                # ViewModels для MVVM паттерна
├── Models/                    # Бизнес-логика приложения
├── Services/                  # Сервисные модули и апи классы
├── Utils/                     # Утилиты и вспомогательные классы
├── Resources/                 # Ресурсы и локализация
├── Config/                    # Файлы конфигурации
├── Docs/                      # Документация
│   ├── README.md             # Этот файл
│   ├── CHANGELOG.md          # История версий
│   └── CONTRIBUTING.md       # Руководство для контрибьюuторов

```

## Технологический стек

- **Язык разработки**: C# 9.0+
- **Фреймворк**: WPF (.NET Framework 4.8)
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Управление зависимостями**: ReactiveUI/Reactive Extensions
- **Локализация**: JSON/XML конфигурации
- **Тестирование**: NUnit/XUnit
- **Система контроля версий**: Git
