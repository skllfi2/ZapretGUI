# Инструкция по созданию иконок для Toast уведомлений

## Требования к иконкам

### Общие спецификации:
- **Формат**: PNG с прозрачностью
- **Размер**: 32x32 пикселя (recommended для toast уведомлений)
- **Цвет**: Белые иконки на прозрачном фоне
- **Стиль**: Простые, чистые и легко узнаваемые

## Описание иконок

### Success.png (Успешное уведомление)
- **Цвет**: Зеленый фон (в иконке должен быть прозрачным)
- **Иконка**: Галочка (✓) или галочка с планецельной обрамляющей линией
- **Размер**: Галочка должна занимать примерно 24x24 пикселя с равной загрузкой в 4 пикселя

### Error.png (Ошибка)
- **Цвет**: Красный фон (в иконке должен быть прозрачным)
- **Иконка**: Крестик или множество X (х размером)
- **Размер**: Иконка должна занимать примерно 24x24 пикселя с равной загрузкой в 4 пикселя

### Warning.png (Предупреждение)
- **Цвет**: Желтый фон (в иконке должен быть прозрачным)
- **Иконка**: Великий треугольник (⚠) или треугольник с точками
- **Размер**: Иконка должна занимать примерно 24x24 пикселя с равной загрузкой в 4 пикселя

## Как создать иконки

### Варианты:
1. **Графические редакторы**: Adobe Photoshop, GIMP, Paint.NET
2. **Онлайн-сервисы**: favicon.io, RealFaviconGenerator
3. **Иконочные библиотеки**: Font Awesome, Material Icons

### Пошаговая инструкция:
```bash
# Для создания иконок с помощью ImageMagick
magick -size 32x32 xc:none -fill "#4CAF50" -draw "circle 16,16 16,4" -fill white -draw "path 'M 12,16 L 20,16 M 16,12 L 16,20'" Success.png

magick -size 32x32 xc:none -fill "#F44336" -draw "circle 16,16 16,4" -fill white -draw "path 'M 12,16 L 20,16 M 16,12 L 16,20'" Error.png

magick -size 32x32 xc:none -fill "#FF9800" -draw "circle 16,16 16,4" -fill white -draw "path 'M 16,12 L 16,20 M 12,16 L 20,16'" Warning.png
```

## Интеграция в проект

1. Скопируйте иконки в папку `Assets/Toast/`
2. Добавьте их в проект Visual Studio
3. Установите свойство "Copy to Output Directory" = "Copy always"

## Проверка

После создания иконок запустите приложение и убедитесь, что:
- Иконки появляются в проекте
- Уведомления отображаются с правильными иконками
- Изменение настроек в настройках уведомлений работают корректно
