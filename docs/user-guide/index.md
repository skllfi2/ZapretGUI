## 🔍 Обзор приложения

**ZapretGUI** — это графический интерфейс для управления системой фильтрации интернет-трафика zapret. Приложение позволяет:

- ✅ **Управлять защитой** — запускать/останавливать систему фильтрации
- ✅ **Конфигурировать стратегии** — выбирать оптимальные настройки для вашего провайдера
- ✅ **Мониторить статус** — отслеживать текущее состояние защиты
- ✅ **Просматривать логи** — анализировать системные сообщения
- ✅ **Проводить тестирование** — проверять работоспособность системы
- ✅ **Управлять обновлениями** — поддерживать актуальность системы

---

## 🎨 Интерфейс приложения

### 🏠 Главное окно

Главное окно приложения представляет собой современный интерфейс с [Mica backdrop](https://docs.microsoft.com/en-us/windows/apps/design/style/mica), который обеспечивает прозрачность и элегантный внешний вид.

![Главное окно ZapretGUI](https://via.placeholder.com/600x400?text=ZapretGUI+Main+Window)

### 📋 Навигация

Навигация реализована с помощью бокового меню **NavigationView** с четырьмя основными разделами:

| Иконка | Раздел | Назначение |
|--------|--------|------------|
| 🏠 | Главная | Обзор статуса системы |
| 📚 | Стратегии | Управление конфигурациями |
| ⌨️ | Обновления | Проверка и установка обновлений |
| 📄 | Логи и тестирование | Анализ системных сообщений |

### 🔗 Основные элементы

#### 1. **Карточка статуса**
- **Индикатор** — круговой индикатор состояния (красный/зеленый)
- **Текст статуса** — "Остановлено" или "Активно"
- **Текущая стратегия** — информация о применяемой конфигурации
- **CoreControl** — графический элемент управления (якр)

#### 2. **Карточки сервисов**
Три карточки отображают статус обхода DPI для популярных сервисов:
- **Discord** — игровые сервисы
- **YouTube** — видеохостинг
- **Google** — поисковая система

#### 3. **Системная строка**
Приложение интегрируется со [строкой задач Windows](https://docs.microsoft.com/en-us/windows/apps/design/shell/tray-notifications), позволяя:
- Минимизировать в трей
- Быстро запускать/останавливать защиту
- Получать уведомления

## ⚡ Основные функции

### 🚀 Запуск/остановка защиты

#### 🎯 Основной метод
1. **Нажмите на CoreControl** — центральный графический элемент
2. **Дождитесь инициализации** — система запускает необходимые службы
3. **Проверьте статус** — индикатор должен стать зеленым

#### ⚠️ Альтернативные способы
- **Через системный трей** — правый клик по иконке
- **Автоматически** — при включенном автозапуске

### 🔧 Стратегии подключения

#### 📖 Что такое стратегии?
Стратегии — это преднастроенные конфигурации для работы с разными провайдерами и сетевыми условиями.

#### 🔍 Типы стратегий
- **Базовые** — стандартные настройки для большинства провайдеров
- **Продвинутые** — оптимизированные конфигурации
- **Пользовательские** — созданные вручную

#### 🎯 Рекомендации
- **Для начала** — используйте базовую стратегию
- **При проблемах** — попробуйте другие варианты
- **Для экспертов** — создайте собственную конфигурацию

### 🔄 Обход DPI

#### 📊 Что такое DPI?
[Глубокая пакетная инспекция](https://ru.wikipedia.org/wiki/Глубокая_пакетная_инспекция) — технология, используемая провайдерами для блокировки трафика.

#### 🔗 Как работает обход
1. **Анализ трафика** — приложение определяет тип пакетов
2. **Применение фильтров** — используется соответствующая стратегия
3. **Мониторинг** — отслеживается статус обхода

### 🔄 Обновления

#### ✅ Автоматическая проверка
- **При запуске** — приложение проверяет наличие обновлений
- **Уведомление** — отображается баннер при обнаружении обновлений
- **Ручная проверка** — вкладка "Обновления"

#### 📦 Процесс обновления
1. **Скачивание** — получение новой версии
2. **Установка** — применение обновлений
3. **Перезапуск** — для применения изменений

### 📝 Логи и тестирование

#### 📊 Логи
- **Системные сообщения** — информация о работе приложения
- **Ошибки** — уведомления о проблемах
- **Отладка** — детальная информация для разработчиков

#### 🧪 Тестирование
- **Скрипты** — готовые тестовые сценарии
- **Интерактивный режим** — ввод данных пользователем
- **Результаты** — отображение в реальном времени

## ⚙️ Настройка и конфигурация

### 🚪 Автозапуск

#### 📋 Варианты автозапуска
| Настройка | Описание | Рекомендация |
|-----------|----------|--------------|
| Автозапуск вместе с Windows | Приложение запускается при входе в систему | ✅ Включить |
| Запускать защиту автоматически | Система фильтрации запускается вместе с приложением | ✅ Включить |
| Минимизация в трей при запуске | Приложение запускается в фоновом режиме | ✅ Включить |

#### 🔧 Как настроить
1. **Откройте вкладку "Настройки"** — через боковое меню
2. **Найдите раздел "Запуск"** — первые параметры
3. **Включите нужные опции** — переключите тумблеры
4. **Сохраните изменения** — настройки применяются автоматически

### 🎨 Тема и язык

#### 🎨 Тема интерфейса
- **Системная** — соответствует настройкам Windows
- **Светлая** — яркая тема для дневного использования
- **Темная** — темная тема для комфортной работы в темное время суток

#### 🌍 Язык интерфейса
- **Русский** — основной язык разработки
- **English** — международная версия

#### 📋 Как изменить
1. **Перейдите в "Настройки"** → "Внешний вид"
2. **Выберите нужную тему** — из выпадающего списка
3. **Выберите язык** — из списка доступных языков
4. **Приложение перезапустится** — для применения изменений

### 📢 Уведомления

#### 💥 Типы уведомлений
| Тип | Описание | Когда используется |
|-----|----------|-------------------|
| Звуковые эффекты | Звуковые сигналы при изменении статуса | ✅ Включить |
| Всплывающие уведомления | Визуальные оповещения | ✅ Включить |
| Уведомления в трее | Иконки в системной строке | ✅ Всегда |

#### 🔧 Как настроить
1. **Откройте "Настройки"** → "Уведомления"
2. **Включите/выключите опции** — переключите тумблеры
3. **Проверьте работу** — измените статус защиты

### 🔍 Фильтры

#### 🎮 Game Filter
**Назначение** — расширить обход DPI на игровые порты (1024-65535)

| Режим | Описание | Рекомендация |
|-------|----------|--------------|
| Отключён | Фильтрация не применяется | ❌ Только для тестирования |
| TCP и UDP | Обход для обоих протоколов | ✅ Для игр |
| Только TCP | Обход только для TCP | ✅ Для стриминга |
| Только UDP | Обход только для UDP | ⚠️ Для голосовой связи |

#### 📧 IPSet Filter
**Назначение** — управление списком IP-адресов для фильтрации

| Режим | Описание | Использование |
|-------|----------|---------------|
| Любой IP (any) | Фильтрация всех IP | ✅ По умолчанию |
| Только из списка (loaded) | Фильтрация только загруженных IP | ⚠️ Экспертный режим |
| Никакой (none) | Без фильтрации IP | ❌ Не рекомендуется |

### 🖥️ Служба Windows

#### 📋 Основные функции
| Функция | Назначение | Рекомендация |
|---------|------------|--------------|
| Установить службу | Установка zapret как службы Windows | ✅ Для постоянной работы |
| Автозапуск службы | Запуск службы при старте Windows | ✅ Для серверов |
| Удалить службы | Удаление всех служб zapret | ❌ Только при проблемах |

#### 🔧 Как управлять
1. **Откройте "Настройки"** → "Служба Windows"
2. **Проверьте статус** — система покажет текущее состояние
3. **Установите/удалите** — нажмите соответствующую кнопку
4. **Проверьте логи** — проанализируйте результаты

## 🔄 Управление стратегиями

### 🎯 Выбор стратегии

#### 📋 Процесс выбора
1. **Откройте вкладку "Стратегии"** — через боковое меню
2. **Просмотрите список** — слева отображаются доступные стратегии
3. **Выберите стратегию** — кликните по названию
4. **Просмотрите информацию** — справа отображаются детали

#### 📊 Информация о стратегии
| Параметр | Описание | Пример |
|----------|----------|---------|
| Тип стратегии | Категория конфигурации | "Базовая", "Продвинутая" |
| Применимость | Условия использования | "Для всех провайдеров" |
| Описание | Детальная информация | "Оптимально для Ростелеком" |

### 🛠️ Применение стратегии

#### 📋 Шаги применения
1. **Выберите стратегию** — из списка
2. **Проверьте информацию** — убедитесь в правильности выбора
3. **Нажмите "Применить стратегию"** — кнопка внизу
4. **Дождитесь подтверждения** — система применит настройки
5. **Проверьте статус** — убедитесь в успешном применении

#### ⚠️ Важные замечания
- **Требуется перезапуск** — после применения стратегии
- **Проверьте соединение** — убедитесь в работоспособности
- **Сохраните настройки** — изменения применяются автоматически

### 📝 Создание собственных стратегий

#### 📋 Процесс создания
1. **Изучите существующие стратегии** — для понимания структуры
2. **Подготовьте конфигурационные файлы** — в формате zapret
3. **Поместите файлы в папку стратегий** — обычно `Strategies/`
4. **Перезапустите приложение** — для обновления списка
5. **Проверьте работу** — убедитесь в корректности стратегии

#### 📚 Требования к стратегиям
- **Формат** — совместим с zapret
- **Валидация** — файлы должны быть корректными
- **Документация** — включите описание стратегии
- **Тестирование** — проверьте перед применением
